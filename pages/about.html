Latest Project

Carolina Hurricanes Warriors Website

The Carolina Hurricanes Warriors Program is a non-profit disabled veteran's ice hockey team. As member of the program I volunteered to develop the teams website to help promote the importance of the program. Currently a new version of the site is under review by the programs board before release.

Originally developed using React and hosted via Heroku. The new and improved site has been now developed using the Vue.js framework using Vuetify components and hosted via AWS Amplify. The site now provides specific calendar events, photos of past events, and a board member login to view team-specific details. Please review the Github page to view more in-depth specifics!

Technologies used include:

  • Vue.js
  • Vuetify
  • AWS Amplify
  • AWS DataStore
  • GraphQl
  • AWS S3
